Add custom sections in the middle of posts or pages using shortcodes.

== Description ==

This plugin allows you to create custom sections that can be inserted into posts or pages using shortcodes. It provides an easy-to-use admin interface to manage sections and their content.

== Installation ==

1. Upload the `custom-middle-section` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the 'Custom Middle Section' tab in the admin panel to create and manage sections.

== Frequently Asked Questions ==

= How do I use the shortcode? =
Copy the shortcode from the admin page and paste it into any post or page where you want the custom section to appear.
